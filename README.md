## 🚀 Key Projects

### 🔍 Lumi Ascend — Automated Ingestion Validator
Polls UDM Audit Tables to detect new ingestions, then compares direct source vs hydration tables. Generates ingestion summary and discrepancy reports. Supports configurable auto-validation thresholds and manual backfill triggers.
> `BigQuery` · `Airflow` · `Python` · `GCS`

### ✅ BQ-to-BQ DVT — Cross-Dataset Validator
BigQuery-native validator for cross-dataset and cross-table comparisons. Enables robust post-migration and intra-cloud validations with schema and data integrity checks.
> `BigQuery` · `Python` · `GCP`

### 🔐 Cross-Env Schema Validator
Automated cross-environment table comparison application that validates schema consistency and data integrity between on-premises Hive and GCP BigQuery. Improved data ingestion efficiency by **80%**.
> `Hive` · `BigQuery` · `Pub/Sub` · `Python`
